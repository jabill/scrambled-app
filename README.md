## About Scrambled

Scrambled is a fast-paced word puzzle game where you unscramble letters before time runs out. Test your vocabulary, challenge yourself with daily puzzles, and track your progress across multiple difficulty levels.

### Key Features

- 🎯 **13,450+ Curated Words** - Carefully validated from trusted dictionaries
- 📅 **Daily Challenge** - Compete globally with the same word each day
- 🏆 **Multiple Modes** - Normal, Expert, and Practice modes
- ⚡ **Smart Scoring** - Shorter timers = bigger multipliers
- 📊 **Progress Tracking** - Stats, streaks, and best scores
- 🔒 **Privacy-First** - No data collection, everything stays on your device
- 🎮 **100% Free** - No ads, no in-app purchases

## Website

This repository hosts the official website for Scrambled, built with:

- **Single-page design** - Fast, responsive, and accessible
- **Dark theme** - Matches the app's design language
- **No dependencies** - Pure HTML, CSS, and JavaScript
- **Mobile-first** - Optimized for all screen sizes

### Local Development

To run the site locally:

```bash
# Clone the repository
git clone https://github.com/jabill/scrambled-app.git

# Navigate to the directory
cd scrambled-app

# Open in your browser
open index.html
# or simply double-click index.html
```

No build process required - it's just static HTML!
